# FastMiner

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Fabric](https://img.shields.io/badge/Mod%20Loader-Fabric-blue)](https://fabricmc.net/)

A **Minecraft Fabric 1.21.2** mod that increases mining speed by **50% (1.5x)** for all tools.

## ✨ Features

- ⛏️ **Increased mining speed** — all tools work 50% faster
- 🎮 **Lightweight** — adds no new blocks or items
- 🔧 **Easy installation** — just download and play

## 📥 Installation

1. Install [Fabric](https://fabricmc.net/use/installer/) for Minecraft 1.21.2
2. Download [Fabric API](https://modrinth.com/mod/fabric-api)
3. Download the latest **FastMiner** from [Releases](https://github.com/kumongul/fastminer-mc/releases)
4. Place the `.jar` files into your `mods` folder
5. Launch the game

## 🛠️ Building from source

```bash
# Clone the repository
git clone https://github.com/kumongul/fastminer-mc.git
cd fastminer-mc

# Build the mod
./gradlew build

# The built .jar file will be in build/libs/
